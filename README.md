# e-plantShopping

Paradise Nursery Shopping Cart Application

## Overview

This is a React-based shopping cart application for Paradise Nursery, an online plant shop offering a variety of house plants. The application features a landing page, product listing with multiple categories, and a fully functional shopping cart with Redux state management.

## Features

- Landing page with background image and "Get Started" button
- Navigation bar with links to Home, Plants, and Cart
- Product listing page with multiple categories:
  - Air Purifying Plants
  - Aromatic Fragrant Plants
  - Insect Repellent Plants
  - Medicinal Plants
  - Low Maintenance Plants
- Add to Cart functionality with button disabling after adding
- Shopping Cart with:
  - Total cart amount calculation
  - Quantity increment/decrement
  - Item removal
  - Continue Shopping and Checkout buttons

## Technology Stack

- React
- Redux Toolkit
- Vite
- CSS

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

4. Preview production build:
   ```bash
   npm run preview
   ```

## Deployment

The application is configured for deployment to GitHub Pages. To deploy:

```bash
npm run deploy
```
